# Simple-Front-End-for-a-General-Knowledge-Quiz-App

This is a simple front-end for a general knowledge quiz app. It was made using HTML, CSS and JavaScript. There is a single question bank made of JSON containing 20 questions. When in play, the game randomises which question to display until 20 questions have been answered. 

Included:

1. quiz_game.html – the HTML file.
2. quiz_game_script.js – the JavaScript file.
3. quiz_game_style.css – the CSS file.

Instructions:

1. Download
2. Open quiz_game.html in a web browser.
3. Press “play” button.
4. Click on an option for your answer.
5. Click “next” button.
6. Continue until round is over.
